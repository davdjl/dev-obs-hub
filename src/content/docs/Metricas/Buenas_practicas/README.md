# Buenas practicas
El tema de las metricas es un el metodo más simple de recolección de datos ya que no requieren un procesamiento o limpieza de la información, el elemento más importante es la periodicidad con la que se recolectan las metricas, en general cada 10 segundos es un tiempo recomendado.
Los elementos más importantes para la correlación de los pilares de observabilidad son los campos de tiempo y nombre de host.
